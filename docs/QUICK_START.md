# Quick Start Guide

## Installation & Running

1. **Navigate to the project:**
   ```bash
   cd financial-hub
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Open your browser:**
   Visit [http://localhost:3000](http://localhost:3000)
